Detailed Description

Using SV_VertexID Demo It is possible to have the input assembler generate a vertex ID.

To do this, add an additional parameter of type uint to the vertex shader signature with semantic SV_VertexID: VertexOut VS(VertexIn vin, uint vertID : SV_VertexID)
