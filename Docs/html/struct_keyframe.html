Detailed Description

A Keyframe defines the bone transformation at an instant in time.summary> A BoneAnimation is defined by a list of keyframes.

/summary> For time values inbetween two keyframes, we interpolate between the two nearest keyframes that bound the time.

We assume an animation always has two keyframes. /summary>
