\doxysubsection{Detailed Description}
Illustrates the minimal amount of the Win32 code needed for Direct3D programming. 

Windows is about two things\+: windows (register window class, create an instance) and messages. Include the windows header file; this has all the Win32 API structures, types, and function declarations we need to program Windows.
