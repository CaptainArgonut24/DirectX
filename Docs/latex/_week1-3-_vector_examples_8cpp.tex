\doxysubsection{Detailed Description}
using vector operations Demo 


\begin{DoxyEnumerate}
\item Use XMVECTOR for local or global variables.\+XMVECTOR needs to be 16-\/byte aligned. This is automatically done for local and global variables.
\item Use XMFLOAT2, XMFLOAT3, and XMFLOAT4 for class data members.
\item Use loading functions to convert from XMFLOATn to XMVECTOR before doing calculations.
\item Do calculations with XMVECTOR instances.
\item Use storage functions to convert from XMVECTOR to XMFLOATn.
\item The calling convention annotation XM\+\_\+\+CALLCONV must be specified before the function name so that the proper calling convention is used
\end{DoxyEnumerate}
