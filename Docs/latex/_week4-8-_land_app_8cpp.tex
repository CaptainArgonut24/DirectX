\doxysubsection{Detailed Description}
Land Demo. 

This demo constructs a triangle grid mesh procedurally and offsets the vertex heights to create a terrain. The graph of a function y = f(x, z) is a surface. We can approximate the surface by constructing a grid in the xz-\/plane, where every quad is built from two triangles, and then applying the function to each grid point

Controls\+: Hold down \textquotesingle{}1\textquotesingle{} key to view scene in wireframe mode. Hold the left mouse button down and move the mouse to rotate. Hold the right mouse button down and move the mouse to zoom in and out.
