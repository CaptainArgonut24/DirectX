\doxysubsection{Detailed Description}
Blend Demo-\/\+No\+Color\+Write! 

Suppose that we want to keep the original destination pixel exactly as it is and not overwrite it or blend it with the source pixel currently being rasterized.\+This can be useful, for example, if you just want to write to the depth / stencil buffer, and not the back buffer.

No Color Write\+: To do this, set the source pixel blend factor to D3\+D12\+\_\+\+BLEND\+\_\+\+ZERO, the destination blend factor to D3\+D12\+\_\+\+BLEND\+\_\+\+ONE, and the blend operator to D3\+D12\+\_\+\+BLEND\+\_\+\+OP\+\_\+\+ADD.

transparency\+Blend\+Desc.\+Src\+Blend = D3\+D12\+\_\+\+BLEND\+\_\+\+ZERO; transparency\+Blend\+Desc.\+Dest\+Blend = D3\+D12\+\_\+\+BLEND\+\_\+\+ONE; transparency\+Blend\+Desc.\+Blend\+Op = D3\+D12\+\_\+\+BLEND\+\_\+\+OP\+\_\+\+ADD,

Controls\+: Hold down \textquotesingle{}1\textquotesingle{} key to view scene in wireframe mode. Hold the left mouse button down and move the mouse to rotate. Hold the right mouse button down and move the mouse to zoom in and out.
