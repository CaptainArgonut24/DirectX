\doxysubsection{Detailed Description}
Shape Practice. 

Place all of the scene geometry in one big vertex and index buffer. Then use the Draw\+Indexed\+Instanced method to draw one object at a time ((as the world matrix needs to be changed between objects)

Controls\+: Hold down \textquotesingle{}1\textquotesingle{} key to view scene in wireframe mode. Hold the left mouse button down and move the mouse to rotate. Hold the right mouse button down and move the mouse to zoom in and out.
