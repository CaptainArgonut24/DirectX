\doxysubsection{Detailed Description}
XMVerify\+CPUSupport Demo. 

\mbox{\hyperlink{namespace_direct_x}{DirectX}} Math is a 3D math library for Direct3D application that is part of the Windows SDK.\+The library uses the SSE2(Streaming SIMD Extensions 2) instruction set.\+With 128 -\/ bit wide SIMD(single instruction multiple data) registers, SIMD instructions can operate on four 32 -\/ bit floats or ints with one instruction.

\begin{DoxyNote}{Note}
use XMVerify\+CPUSupport to check support for SSE2 (Pentium4, AMD K8, and above).
