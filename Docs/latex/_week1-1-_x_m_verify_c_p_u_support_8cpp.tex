\doxysubsection{Detailed Description}
XMVerify\+CPUSupport Demo. 

\doxyref{DirectX}{p.}{namespace_direct_x} Math is a 3D math library for Direct3D application that is part of the Windows SDK.\+The library uses the SSE2(Streaming SIMD Extensions 2) instruction set.\+With 128 -\/ bit wide SIMD(single instruction multiple data) registers, SIMD instructions can operate on four 32 -\/ bit floats or ints with one instruction.

\begin{DoxyNote}{Note}
use XMVerify\+CPUSupport to check support for SSE2 (Pentium4, AMD K8, and above). 
\end{DoxyNote}
\begin{DoxyAttention}{Attention}
use \#include \texorpdfstring{$<$}{<}windows.\+h\texorpdfstring{$>$}{>} for XMVerify\+CPUSupport
\end{DoxyAttention}
In addition, for the x86 platform you should enable SSE2 (Project Properties \texorpdfstring{$>$}{>} Configuration Properties \texorpdfstring{$>$}{>} C/\+C++ \texorpdfstring{$>$}{>} Code Generation \texorpdfstring{$>$}{>} Enable Enhanced Instruction Set), and for all platforms you should enable the fast floating point model /fp\+:fast (Project Properties \texorpdfstring{$>$}{>} Configuration Properties \texorpdfstring{$>$}{>} C/\+C++ \texorpdfstring{$>$}{>} Code Generation \texorpdfstring{$>$}{>} Floating Point Model). You do not need to enable SSE2 for the x64 platform because all x64 CPUs support SSE2
