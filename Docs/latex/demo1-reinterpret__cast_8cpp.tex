\doxysubsection{Detailed Description}
reinterpret\+\_\+cast is a type of casting operator used in C++. 

It is used to convert a pointer of some data type into a pointer of another data type, even if the data types before and after conversion are different. It does not check if the pointer typeand data pointed by the pointer is same or not. Syntax \+:

data\+\_\+type $\ast$ var\+\_\+name = reinterpret\+\_\+cast \texorpdfstring{$<$}{<}data\+\_\+type$\ast$\texorpdfstring{$>$}{>}(pointer\+\_\+variable); Return Type

It doesn’t have any return type.\+It simply converts the pointer type.
