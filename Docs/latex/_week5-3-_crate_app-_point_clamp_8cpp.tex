\doxysubsection{Detailed Description}
\mbox{\hyperlink{struct_texture}{Texture}} Demo using Point Filtering and Clamp Address Mode ! \mbox{\hyperlink{class_a}{A}} texture, combined with constant or linear interpolation, defines a vector-\/valued function T(u, v) = (r, g, b, a). 

That is, given the texture coordinates (u, v) âˆˆ \mbox{[}0, 1\mbox{]}2 the texture function T returns a color (r, g, b, a). Direct3D allows us to extend the domain of this function in four different ways (called address modes)\+: wrap, border color, clamp, and mirror.
\begin{DoxyEnumerate}
\item wrap extends the texture function by repeating the image at every integer junction
\item border color extends the texture function by mapping each (u, v) not in \mbox{[}0, 1\mbox{]} to some color specified by the programmer
\item clamp extends the texture function by mapping each (u, v) not in \mbox{[}0, 1\mbox{]} to the color T(u0, v0), where (u0, v0) is the nearest point to (u, v) contained in \mbox{[}0, 1\mbox{]}
\item mirror extends the texture function by mirroring the image at every integer junction
\end{DoxyEnumerate}
